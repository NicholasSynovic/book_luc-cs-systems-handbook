##########
 Overview
##########

Dijkstra is the Loyola University Chicago (LUC) Computer Science (CS)
Department's primary high-performance cluster computer. It is intended
to provide educators and students with a standardized platform to teach
Linux, systems, distributed and high-performance computing, and
artificial intelligence techniques. For academic researchers, it
provides a job scheduling interface allowing for jobs to leverage entire
or multiple nodes concurrently while ensuring that resources are not
concurrently being shared between users.

Dijkstra is a *heterogenous* system; Dijkstra nodes are not necessarily
the same hardware or system architecture. This is a feature, not a bug!
The heterogenity of this system allows for researchers to easily add
nodes of different architectures (e.g., arm, risc-v) and leverage them
through Dijkstra's job scheduling system.
